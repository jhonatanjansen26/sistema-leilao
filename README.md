# Sistema de Casa de Leilões

## Explicação do Projeto
Este projeto trata-se de um sistema informatizado desenvolvido para uma casa de leilões. O objetivo é modernizar e gerenciar as rotinas diárias, garantindo maior controle, segurança e eficiência nas operações de leilão utilizando práticas modernas de qualidade de software.

## Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** MySQL 
* **Versionamento:** Git e GitHub
